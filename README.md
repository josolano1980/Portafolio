## 📋 Características

### ✨ Diseño Moderno
- Interfaz moderna y profesional
- Tema oscuro con gradientes azules y cian
- Animaciones fluidas y efectos visuales
- Diseño responsivo (Mobile-First)

### 🎯 Secciones
1. **Navegación Fija** - Menú sticky con links suave
2. **Hero Section** - Presentación impactante con CTA

6. **Contacto** - Formulario y información de contacto
7. **Footer** - Enlaces sociales

### 🔧 Stack Tecnológico
- **HTML5** - Estructura semántica

## 🎨 Personalización

### Colores Principales (Variables CSS)
```css
--primary-color: #0066ff;      /* Azul primario */
--secondary-color: #00d4ff;    /* Cian */
--accent-color: #ff6b35;       /* Naranja acentos */
--dark-bg: #0a0e27;            /* Fondo oscuro */
```

### Modificar Información
1. **Datos Personales**: Actualiza en la sección de contacto
2. **Proyectos**: Agrega/edita en la sección de proyectos
3. **Competencias**: Personaliza las categorías y habilidades
4. **Redes Sociales**: Actualiza enlaces en el footer
